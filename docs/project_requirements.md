# Customer Churn Prediction System

## Business Objective

Predict whether a customer will leave the company.

## Stakeholders

- Management
- Marketing Team
- Customer Retention Team

## Expected Output

Prediction:
- Churn
- No Churn

Probability Score:
- 0-100%

## Success Criteria

- F1 Score > 85%
- User-friendly web interface
- Cloud deployment

## Risks

- Imbalanced dataset
- Missing values
- Data quality issues
